# Java Project: Reverse-and-Add Palindromes Detection

This program was written as part of the "Grundlagen der Programmierung" course at Humboldt University of Berlin.

It implements a variant of the Reverse-and-Add algorithm and attempts to detect numbers that do not reach a palindrome within a reasonable number of iterations.

## Original Assignment

The assignment description is provided in `A02_WS2223.pdf` (in German).

## How to Compile and Run

```bash
javac Nolindrom.java
java Nolindrom 300
```
